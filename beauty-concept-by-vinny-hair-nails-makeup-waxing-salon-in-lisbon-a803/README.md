# Beauty Concept by Vinny Demo

Static Next.js export for the GitHub Pages demo URL:

https://barobaonguyen.github.io/demos/beauty-concept-by-vinny-hair-nails-makeup-waxing-salon-in-lisbon-a803/

## Palette

- Soft cream base: `#faf6f1`
- Warm paper: `#fffdf9`
- Ink: `#251d1a`
- Rose-gold accent: `#c08a72`
- Soft rose accent: `#d9a59a`
- Warm taupe: `#b8a79a`
- Muted sage: `#78806f`

## Build

```bash
npm run build
```

The site uses `output: 'export'` and writes static files to `out/`.
