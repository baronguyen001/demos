# Luis Rubio Barber Shop Demo

Static one-page cold-outreach demo for Luis Rubio Barber Shop in Barrio de
Salamanca, Madrid.

## Stack

- Next.js App Router with TypeScript
- Tailwind CSS
- Static export with `output: 'export'`
- GitHub Pages base path: `/demos/luis-rubio-barber-shop-a870`

## Palette

- Deep charcoal base: `#1a1a1a`
- Soft charcoal section base: `#232323`
- Warm gold accent: `#c8a04d`
- Warm off-white page base: `#f7f6f1`
- White surface: `#ffffff`

## Build

```bash
npm run build
```

The static export is generated in `out/`. For GitHub Pages, flatten `out/*` into
this folder and remove `out/`.
